# Kebijakan Privasi - Aplikasi Lawang

**Aplikasi:** Lawang - Aplikasi Statistik Indonesia  
**Pengembang:** Badan Pusat Statistik (BPS) Republik Indonesia  
**Versi:** 1.0  
**Tanggal Efektif:** [Tanggal Publikasi]  
**URL:** [https://www.bps.go.id/privacy-policy/lawang]

---

## 1. Pendahuluan

Badan Pusat Statistik (BPS) berkomitmen untuk melindungi privasi pengguna aplikasi Lawang. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, menyimpan, dan melindungi informasi pengguna.

## 2. Informasi yang Kami Kumpulkan

### 2.1 Informasi yang Dikumpulkan Secara Otomatis
- **Informasi Perangkat:** Jenis perangkat, versi sistem operasi, identifikasi perangkat unik (non-personal)
- **Informasi Penggunaan:** Statistik penggunaan aplikasi, fitur yang diakses, waktu penggunaan
- **Lokasi:** Data lokasi geografis (jika pengguna memberikan izin) untuk fitur peta
- **Log Error:** Informasi teknis untuk diagnosa masalah aplikasi

### 2.2 Informasi yang Tidak Kami Kumpulkan
Aplikasi Lawang **TIDAK** mengumpulkan:
- Informasi identitas pribadi (nama, NIK, alamat, email, nomor telepon)
- Data sensitif (keuangan, kesehatan, biometrik)
- Data dari kontak, galeri, atau file perangkat lainnya

## 3. Penggunaan Informasi

Informasi yang dikumpulkan digunakan untuk:
- Meningkatkan performa dan fitur aplikasi
- Menganalisis pola penggunaan untuk pengembangan lebih lanjut
- Memberikan informasi statistik yang relevan berdasarkan lokasi
- Mendiagnosis dan memperbaiki masalah teknis
- Menghasilkan statistik penggunaan aplikasi secara anonim

## 4. Penyimpanan dan Keamanan Data

- Semua data disimpan secara lokal di perangkat pengguna
- Tidak ada data pribadi yang dikirim ke server eksternal
- Data statistik yang dianalisis bersifat anonim dan teraggregasi
- Kami menggunakan enkripsi standar industri untuk melindungi data

## 5. Berbagi Informasi

BPS **TIDAK** menjual, menyewakan, atau membagikan data pengguna kepada pihak ketiga.

Data hanya dapat dibagikan dalam situasi berikut:
- Jika diwajibkan oleh hukum atau perintah pengadilan
- Untuk melindungi hak, properti, atau keselamatan BPS atau publik

## 6. Hak Pengguna

Pengguna memiliki hak untuk:
- Mengakses data statistik yang tersedia
- Menonaktifkan fitur lokasi melalui pengaturan perangkat
- Menghapus aplikasi dan semua data lokal yang tersimpan
- Mengajukan pertanyaan atau keluhan terkait privasi

## 7. Izin Aplikasi

Aplikasi Lawang meminta izin berikut:
- **Akses Internet:** Untuk mengunduh data statistik terbaru
- **Lokasi:** Untuk fitur peta dan informasi regional (opsional)
- **Penyimpanan:** Untuk menyimpan data statistik secara lokal
- **Status Jaringan:** Untuk memeriksa konektivitas internet

## 8. Data Anak-anak

Aplikasi ini tidak secara spesifik ditujukan untuk anak-anak di bawah 13 tahun. BPS tidak dengan sengaja mengumpulkan data pribadi dari anak-anak.

## 9. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan diumumkan melalui aplikasi dan/atau website BPS.
